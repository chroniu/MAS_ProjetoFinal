Instruções para execução.

1. Instalar o Anaconda(distribuição python)
2. Clonar o repositório para a máquina local
3. Instalar o ambiente virtual que contem todos os pacotes utilizados no projeto(pode demorar alguns minutos para terminar a instalação)

No terminal, na pasta do projeto 
>> conda env create -f environment.yml
>> conda activate mas4

4. Agora para rodar o projeto, no terminal:
(mas4) >> python3 mas_fase01.py
(mas4) >> python3 mas_fase02.py


5. O arquivo database.py contêm o acesso ao banco de dados. Alterar a senha/usuário/database para os da máquina local, ou comentar a linha e descomentar a anterior para usar o sqlite3. O problema do sqlite3 é que é muito lento para rodar real. 

5. Escolar a Fase 01 e/ou Fase 02  e executar

conda install -n mas  dataset



Para rodar o applicativo/gui, no terminal
>> bokeh serve --show masapp
