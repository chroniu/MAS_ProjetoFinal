Instruções para execução.

1. Instalar o Anaconda(distribuição python)
2. Clonar o repositório para a máquina local
3. Instalar o ambiente virtual que contem todos os pacotes utilizados no projeto(pode demorar alguns minutos para terminar a instalação)

No terminal, na pasta do projeto 
>> conda install --name mas3 --file environment.yml

  Depois de instalado rodar o script "start_project.sh" -> basicamente só cria pastas e arquivos de log, 
se for rodar no windows, é só criar as pastas e arquivos com os mesmos nomes do script.


3. O arquivo database.py e masapp/database.py contêm o acesso ao banco de dados. 
Alterar a senha/usuário/database para os da máquina local


4. Agora para rodar a fase 01 ou a fase 02, no terminal:
(mas3) >> python3 mas_fase01.py
(mas3) >> python3 mas_fase02.py

5. Para rodar a fase 3
Para rodar o applicativo/gui, no terminal
(mas3) >> bokeh serve --show masapp
